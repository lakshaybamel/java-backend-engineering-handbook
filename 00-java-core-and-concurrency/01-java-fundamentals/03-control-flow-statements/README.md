## 🧠 Key Takeaways

* if-else -> flexible condition handling
* switch -> cleaner for fixed values
* for loop -> known iterations
* while loop -> condition-based execution
* do-while -> runs at least once
* break -> exit loop
* continue -> skip iteration

---
