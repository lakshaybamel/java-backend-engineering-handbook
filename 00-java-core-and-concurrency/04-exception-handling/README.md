# ⚠️ Exception Handling

This module covers how Java handles **runtime errors and unexpected situations** to maintain smooth program execution.

---

## 📖 Overview

Exception handling helps in:

* Preventing program crashes
* Managing runtime errors
* Maintaining normal flow of execution
* Writing reliable and robust code
