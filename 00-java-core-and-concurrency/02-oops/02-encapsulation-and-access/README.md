# 🔐 Encapsulation and Access Modifiers

This module covers how Java controls access to data using **encapsulation and access modifiers**, ensuring data safety and proper design.

---

## 📖 Overview

Encapsulation is one of the core principles of OOP. It focuses on:

* Restricting direct access to data
* Providing controlled access through methods
* Improving security and maintainability

Access modifiers support encapsulation by defining **who can access what**.

## 🎯 Key Takeaways

* Encapsulation protects data using controlled access
* Access modifiers define visibility of variables and methods
