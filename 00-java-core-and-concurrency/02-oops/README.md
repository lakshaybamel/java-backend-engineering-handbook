# 🧱 Object-Oriented Programming (OOP)

This section covers the core concepts of OOP in Java, which are essential for building **structured, maintainable, and scalable applications**.

---

## 📖 Overview

Object-Oriented Programming helps organize code around real-world entities using:

* Classes and objects to represent data
* Encapsulation to protect and control access
* Inheritance to reuse existing code
* Polymorphism to enable flexible behavior
* Abstraction to hide implementation details
* Design principles to write clean and scalable systems
